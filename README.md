# 에라 도미니언 3D v3

Three.js 기반 3D 웹게임 프로토타입입니다.

## v3 수정 내용

- 게임이 한 번씩 멈추는 문제를 줄이기 위해 게임 재시작 시 이전 렌더러를 정리하도록 수정했습니다.
- NPC 흡수 및 적 세력 간 흡수 로직을 정리했습니다.
- 적 세력도 흡수 범위 안의 중립 시민과 약한 적 세력을 실제로 흡수하며 성장합니다.
- 플레이어와 적, 적과 적의 전투 판정을 흡수 범위 기반으로 재조정했습니다.
- 진화 시 단순 색 변경/크기 변경이 아니라 캐릭터 모델을 새로 그리도록 수정했습니다.
- 단계별 외형을 분리했습니다.
  - 1단계: 기본 시민형
  - 2단계: 도시 대장, 어깨 갑옷과 지팡이
  - 3단계: 구역 지배자, 검과 투구
  - 4단계: 도시 군주, 긴 망토와 문장 링
  - 5단계: 도시의 왕, 왕관과 홀
  - 6단계: 도시의 황제, 대형 왕관, 황금 망토, 오라 장식
- 일반 시민은 이름표를 표시하지 않고, 2단계 이상 특별 개체만 단계명과 별, 전투력을 표시합니다.

## 실행 방법

압축을 풀고 `index.html`을 브라우저에서 열면 됩니다.
GitHub/Vercel 업로드 시에는 ZIP이 아니라 압축을 푼 파일을 저장소 최상위에 올리세요.
