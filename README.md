# 에라 도미니언 v5

## 수정 내용
- 화면 잘림 방지: 플레이 화면이 각 모니터/브라우저 높이에 맞게 자동 축소됩니다.
- 진화 이펙트 조절: 화면을 가리지 않도록 파티클과 배너 강도를 낮췄습니다.
- 단계 구분 강화: 플레이어와 적 위에 현재 단계와 세력 수/위험도가 표시됩니다.
- 외형 차별화: 단계별 헬멧, 왕관, 망토, 어깨 장식, 황제 후광을 추가했습니다.
- 적 세력 판별 강화: 적이 나보다 강하면 “위험”, 약하면 “약함”, 비슷하면 “비슷”으로 표시됩니다.
- 그래픽 개선: 바닥 타일, 음영, 캐릭터 실루엣, UI 광택을 강화했습니다.
- 이동 속도는 v4보다 더 안정적으로 낮췄습니다.

## 업로드 방법
압축을 푼 뒤 `index.html`, `styles.css`, `script.js`, `README.md`를 GitHub 저장소에 덮어쓰기 업로드하세요.
