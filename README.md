# 에라 도미니언 3D v6

그래픽 퀄리티 개선 버전입니다.

## 반영 내용
- 캐릭터 크기와 실루엣 강화
- 단계별 장비, 망토, 방패, 검, 왕관, 보석, 황제 지팡이/오브 디테일 추가
- 고대 마을 맵 디테일 개선: 돌바닥, 도로, 창문 조명, 시장 가판대, 가로등, 분수 디테일
- 조명/그림자 품질 개선: 톤 매핑, 색공간 보정, 그림자 해상도 상향, 림 라이트 추가
- 카메라 각도 조정으로 캐릭터 디테일이 더 잘 보이게 개선
