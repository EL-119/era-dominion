# Era Dominion 3D - Reborn

GitHub/Vercel 배포용 단일 HTML 웹게임입니다.

## 업로드 파일
- index.html
- README.md

assets 폴더 없이 작동하도록 구성했습니다.

## 조작
- 이동: WASD 또는 방향키
- 일시정지: Esc 또는 P
- 모바일: 좌측 하단 가상 조이스틱

## 게임 구조
- 플레이어와 적 리더만 NPC와 약한 세력을 흡수합니다.
- 일반 NPC끼리는 서로 흡수하지 않습니다.
- 리더 단계: 시민 리더 → 도시 대장 → 구역 지배자 → 도시 군주 → 도시의 왕 → 도시의 황제
- 단계가 오를수록 머리, 갑옷, 방패, 검, 투구, 망토, 왕관, 황제 예복이 추가됩니다.
