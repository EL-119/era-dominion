# 에라 도미니언 3D v1

Three.js 기반 3D 웹게임 프로토타입입니다.

## 포함 기능
- 3D 고대 마을 월드맵
- 입체 건물, 나무, 분수, 그림자
- 건물 및 장애물 충돌 처리
- 플레이어 추적 카메라
- 시민 흡수 및 세력 증가
- 성장형 적 세력 AI
- 적과 플레이어 전투력 비교 및 GAME OVER
- 단계별 외형, 이름, 별, 전투력 표시
- 진화 이펙트와 진화 UI
- 미니맵, 랭킹, 점수, 제한시간 HUD
- PC 키보드 및 모바일 조이스틱 지원

## 배포 방법
압축을 풀고 `index.html`을 포함한 파일을 GitHub 저장소에 업로드한 뒤 Vercel에서 Import 하면 됩니다.

## 주의
Three.js CDN을 사용하므로 인터넷 연결이 가능한 배포 환경에서 정상 실행됩니다.
