# 에라 도미니언 v6

수정 내용
- 건물 모서리/벽에 끼이는 현상을 줄이기 위해 이동 충돌을 단계별 처리와 밀어내기 방식으로 개선했습니다.
- 게임 화면이 모니터 크기에 맞춰 자동으로 캔버스 내부 해상도까지 재계산되도록 수정했습니다.
- 플레이어 이동 속도를 더 낮추고 조작감을 안정화했습니다.
- 강한 적에게 밀리면 점수만 1로 떨어지는 것이 아니라 실제 GAME OVER가 발생하도록 수정했습니다.
- 적과 플레이어 위에 단계/위험도/전투력 표시를 더 명확하게 표시했습니다.
- 전체 화면 게임 모드 레이아웃을 다시 조정했습니다.
