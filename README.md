# 에라 도미니언 v7

수정 내용
- 게임 시작하기, 진화 단계 보기, 스테이지 버튼이 화면 상단으로만 이동하던 문제 수정
- 게임 화면을 전체 브라우저 화면에 고정하고 캔버스가 자동으로 비율 맞춤
- 단계별 캐릭터 외형, 왕관, 망토, 문장, 별 표시 강화
- 적 세력의 강약을 게임 화면에서 바로 구별하도록 위험/약함 표시 추가
- 진화 이펙트를 약하게 조절해 캐릭터가 잘 보이도록 개선
- 건물 충돌 및 끼임 방지 로직 개선
- 강한 적에게 닿으면 점수 감소가 아니라 GAME OVER 처리
- 전체 캔버스 그래픽, 건물, 캐릭터 표현 개선
